How to Add the Conference Dropdown

To add the Conference dropdown exactly as it appears in the navigation, copy and paste the following HTML code into your navigation bar (usually in base.html):

<li class="relative group">
  <a href="#" class="hover:text-blue-600 flex items-center">Conference 
    <svg class="w-4 h-4 ml-1" fill="none" stroke="currentColor" stroke-width="2" viewBox="0 0 24 24">
      <path stroke-linecap="round" stroke-linejoin="round" d="M19 9l-7 7-7-7" />
    </svg>
  </a>
  <div class="absolute left-0 mt-2 w-64 bg-white rounded-lg shadow-lg opacity-0 group-hover:opacity-100 group-hover:pointer-events-auto transition-opacity duration-200 z-50 invisible group-hover:visible">
    <ul class="py-2 text-gray-700">
      <li><a href="#" class="block px-6 py-2 hover:bg-blue-50">New Proposal / Terms</a></li>
      <li><a href="#" class="block px-6 py-2 hover:bg-blue-50">Special Issues</a></li>
      <li><a href="#" class="block px-6 py-2 hover:bg-blue-50">Upcoming Conferences</a></li>
    </ul>
  </div>
</li>

Paste this inside your navigation <ul> where you want the Conference dropdown to appear. You can update the links as needed.
